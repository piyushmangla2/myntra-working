True margins — Layer 1 & Layer 2
Net settlement − return cost − COGS = L1. L1 + Rebate (+60d) = L2. Both shown per design.
Two-layer margin model — return-corrected. Verified from settlement data: Myntra credits commission back on returns ✓ · Product comes back so COGS applies only to kept units ✓ · Only real cash cost of a return is reverse logistics (already in settlement). So margin = (net cash − COGS × kept units) ÷ gross GMV shipped. Layer 2 adds MBB rebate (+60 days). COGS is editable — margins recalculate live.
