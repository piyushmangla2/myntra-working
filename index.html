Data Backup & Restore

Data is stored in Supabase cloud — accessible from any device, synced in real-time between Mac, Windows, and mobile. The JSON export is a portable offline backup.

Export All Data
Downloads a complete JSON backup — vendors, SKUs, challans, GRNs, raw materials, cutting orders, routes, everything.
Import Data
Restore from a previously exported JSON file. Replaces all current data — export first if you want to keep it.
Migrate Local Data to Cloud
One-time migration from this browser's old local storage to Supabase. Only use if the cloud is empty or this browser has the most complete data — it will overwrite cloud data.
👥 User Management

Control who can access the app and what they can do. Roles: Admin (full access), Manager (no delete), Operator (no void/delete), Viewer (read only).

📋 Audit Log

Every create, edit, void, delete, payment, and GRN action is logged with the user's name, role, and timestamp. The last 2,000 entries are kept.

💾 Backup & Export

Download a complete backup of all your data. Keep a weekly copy offline — your data lives in Supabase but a local backup gives you an extra safety net.
